# microTCP

## Introduction

A lightweight TCP implementation using UDP transport layer.

* Built for a wide range of battery powered IoT devices
* Provides reliable communication and data transfer

## Features

* Error correction
* Retransmission
* Fast retransmit
* flow control
* congestion control

## Build requirements

To build this project `cmake` is needed.

## Build instructions

```bash
mkdir build
cd build
cmake ..
make
```
